# 自演化知识管理系统

> 让AI从你的简洁笔记中自动学习和构建知识库

## 项目状态

🚧 **设计阶段** - 核心设计文档已完成，准备开始实现

## 快速了解

这个系统解决了什么问题？

**传统方式：**
```markdown
# 你的日志（需要写得很详细）
linux81是公司的内网服务器，配置为8核CPU+64GB RAM，
主要用于运行大模型推理服务。部署了QAMath项目，
该项目是一个基于Qwen-8B的数学问答系统...
```

**新方式：**
```markdown
# 你的日志（保持简洁）
linux81
  QAMath启动正常
```

AI自动学习：
- `linux81`是服务器（从路径推断）
- `QAMath`是项目（从上下文推断）
- 它们之间有"部署"关系（从共现推断）

AI还会主动询问不确定的信息：
> "我经常看到'linux81'，它是内网服务器吗？配置是什么？"

## 核心特性

✅ **渐进式学习** - 从文档和对话中持续学习
✅ **自动补全** - 查询时自动补充背景知识
✅ **冲突检测** - 发现文档中的矛盾信息
✅ **证据追踪** - 每个回答都标注来源
✅ **置信度管理** - 追踪知识的可靠性
✅ **完全本地** - 隐私安全，离线可用
